## Backlink tracking note

Log every submission in affiliate-tracker.md under a new "Directory submissions" tab with these columns: directory name, submission URL, submission date, approval status (pending / approved / rejected), live listing URL, dofollow / nofollow, anchor text, referral traffic in first 30 days (via Plausible). Re-check pending submissions every 7 days. If a directory rejects, ask for feedback and resubmit with the requested changes - rejection is recoverable.

---

## Frequency cap

Do not submit to more than 3 to 4 directories per week. Most directories detect rapid simultaneous submissions and flag for review. Spread submissions over 6 to 8 weeks for Tier 1 + Tier 2. Tier 3 can batch later.

---

## What to do when a listing goes live

1. Confirm the link is dofollow (use Ahrefs or browser extension "NoFollow Simple"). If nofollow when expected dofollow, contact the directory admin once - sometimes settings differ between free and paid tiers.
2. Confirm the anchor text is "MytheAi" or "mytheai.com" - some directories default to "Click here" or "Visit website" which dilutes the link signal. Request descriptive anchor text where editable.
3. Add the listing to a "Live backlinks" Google Sheet with: domain, anchor, dofollow status, first-seen date.
4. Set a calendar reminder to check the listing every 90 days - directories sometimes delist or move to paywall.
5. Re-share the listing on twitter (@mytheai) and LinkedIn (John Pham personal) once it is approved. The social signal compounds the link signal.

## Operational rhythm

Weekly: spend 60 to 90 minutes on directory submissions during the first 8 weeks of the campaign. Aim for 2 to 4 new submissions per week. After week 8, drop to maintenance mode (1 to 2 submissions per month, mostly Tier 3 cleanups and follow-up replies).

Monthly: review affiliate-tracker.md "Directory submissions" tab. Calculate the dofollow link gain, referral traffic per directory, and approval rate. Drop directories with < 5 referral visits per month from future re-submissions.

Quarterly: re-pitch any directory that rejected or never replied, using a new milestone hook (250 hand-tested, new category opened, traffic threshold reached).
